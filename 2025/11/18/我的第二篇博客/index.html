Python 入门：从环境搭建到第一个项目
一、为什么学 Python？
Python 是目前最受欢迎的编程语言之一，原因很简单：
语法简洁：代码像英语句子一样易读，新手入门门槛低
用途广泛：Web 开发、数据分析、人工智能、自动化运维、爬虫等
社区强大：丰富的第三方库（如requests、pandas、TensorFlow）
就业前景好：薪资高、岗位需求多，是编程入门的首选语言
二、环境搭建（Windows/macOS/Linux 通用）

  1. 下载 Python
    官网地址：https://www.python.org/downloads/
    版本选择：推荐 3.8+（避免 2.x 版本，已停止维护）
    安装注意：
    Windows：勾选「Add Python to PATH」（自动添加环境变量）
    macOS/Linux：默认已安装，可通过python3 –version检查版本
  2. 验证安装
    打开命令行（Windows：CMD/PowerShell；macOS/Linux：Terminal），输入以下命令：
    bash
    运行
    python –version # Windows系统
    python3 –version # macOS/Linux系统
    若显示版本号（如Python 3.10.12），则安装成功。
  3. 选择编辑器
    新手推荐：IDLE（Python 自带，简单易用）、VS Code（安装 Python 插件）
    专业开发：PyCharm（社区版免费，功能强大）
    三、第一个 Python 程序：Hello World
  4. 交互式环境运行
    打开命令行，输入python（或python3）进入交互式环境，输入以下代码：
    python
    运行
    print(“Hello, Python!”)
    按回车，会输出：Hello, Python!
    （退出交互式环境：输入exit()或按Ctrl+Z+ 回车）
  5. 脚本文件运行
    新建文件，命名为hello.py（Python 文件后缀为.py）
    写入代码：
    python
    运行

这是注释（#后面的内容不会执行）

name = input(“请输入你的名字：”) # 接收用户输入
print(f”你好，{name}！欢迎学习Python~”) # f-string格式化输出
运行脚本：
bash
运行
python hello.py # Windows
python3 hello.py # macOS/Linux
输入名字后，会输出个性化问候。
四、基础语法速览

  1. 变量与数据类型
    Python 无需声明变量类型，直接赋值即可：
    python
    运行

常用数据类型

num1 = 10 # 整数（int）

input()：接收用户输入（默认返回字符串）
print()：输出内容（支持多参数、格式化）
python
运行

输入年龄并转换为整数

print(f”明年你将{age + 1}岁”) # f-string（推荐，Python3.6+）

for 循环：遍历序列（列表、字符串等）

运行

遍历字符串

print(char)

5. 列表与字典
列表（list）：有序、可修改的集合
python
运行
fruits = [“苹果”, “香蕉”, “橙子”]
print(fruits[0]) # 访问第一个元素：苹果
fruits.append(“葡萄”) # 添加元素
fruits[1] = “猕猴桃” # 修改元素
print(fruits) # 输出：[‘苹果’, ‘猕猴桃’, ‘橙子’, ‘葡萄’]
字典（dict）：无序、键值对集合（键唯一）
python
运行
student = {
“name”: “小红”,
“age”: 18,
“gender”: “女”
}
print(student[“name”]) # 访问值：小红
student[“age”] = 19 # 修改值
student[“score”] = 95 # 添加新键值对
print(student) # 输出：{‘name’: ‘小红’, ‘age’: 19, ‘gender’: ‘女’, ‘score’: 95}
五、简单项目：简易计算器
实现加减乘除功能，结合输入输出、条件判断、循环：
python
运行
print(“简易计算器（输入’q’退出）”)
while True:
# 接收输入
num1 = input(“请输入第一个数字：”)
if num1 == “q”:
break # 退出循环
num2 = input(“请输入第二个数字：”)
if num2 == “q”:
break
operator = input(“请输入运算符（+,-,*,/）：”)

# 转换数字类型
try:
    num1 = float(num1)
    num2 = float(num2)
except ValueError:
    print("输入错误，请输入有效数字！")
    continue

# 计算并输出结果
if operator == "+":
    print(f"{num1} + {num2} = {num1 + num2}")
elif operator == "-":
    print(f"{num1} - {num2} = {num1 - num2}")
elif operator == "*":
    print(f"{num1} * {num2} = {num1 * num2}")
elif operator == "/":
    if num2 == 0:
        print("错误：除数不能为0！")
    else:
        print(f"{num1} / {num2} = {num1 / num2}")
else:
    print("错误：请输入正确的运算符！")

七、总结
Python 入门不难，关键是多写代码、多练项目。建议从简单的小工具（如批量重命名文件、爬取网页内容）开始，逐步积累经验。坚持下去，你会发现 Python 的强大之处！
